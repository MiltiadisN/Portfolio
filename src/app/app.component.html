Employee & Task Management Application

The Employee & Task Management Application is a web-based platform for managing employees and tasks, built with Angular and Spring Boot. It features a responsive UI using Bootstrap, Angular Material, and custom HTML/CSS, and is deployed via Docker on a VPS server with MySQL for secure data storage. The app includes a secure Spring Security login system, allowing managers to log in, view, update, and assign tasks to employees. This project demonstrates full-stack development expertise with seamless CRUD operations for managing employees and tasks.

Greek Restaurant Website

This project showcases a modern and responsive website built for a Greek restaurant located in Altenfurt. This project highlights expertise in full-stack web development, with a focus on responsive design and robust backend systems. The backend, developed using Laravel, empowers the restaurant manager to independently manage the menu with features to add, delete, or update items effortlessly. AWS-Integrated Authentication App

This project demonstrates an Angular application that integrates AWS services such as Amazon Cognito, Lambda, and API Gateway to provide user authentication and authorization features. Users can register, log in, reset passwords, and manage their sessions using Cognito. The UI is built with Bootstrap, and the development and hosting were done on Firebase.

Weather App

This project is a simple weather application built with Angular, allowing users to search for weather details such as temperature, humidity, and wind speed for any city. It uses data from the OpenWeatherMap API and requires an API key to function. The UI enables users to input a city and view weather information quickly. The development and deployment were done on Azure.
